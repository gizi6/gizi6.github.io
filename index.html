Web Designer Yoshiki Kawajiri

富山県在住のWebデザイナー、川尻恭生です。オンラインスクールでWeb制作について学び、大学在学中に興味を持ったWebデザイナーへと就職活動中です。これからさらにWeb制作の技術や知識を身につけ、Webサイト運用や問題解決のアドバイスもできるデザイナーになりたいと思っています。

Skills

Web

HTML/CSSでのコーディングができます。サークルでサイト制作を行った経験があります。現在、javascriptの勉強もしています。

Illust

サークルでポスター作りやキャライラスト制作を行ったことがあります。現在、photoshopについて勉強中です。

Direction

Webディレクションについて勉強しています。お客様の要望をヒアリングし、費用対効化の高いwebサイト制作の土台を作ります。
